Tag de imagem

A tag para exibir uma imagem é a tag "img", que tem atributos como src=, que permite definir o caminho para fonte de imagem

Tag de listagem

O HTML possui listas ordenadas e não ordenadas que podem ser definidas como as tags ol e ul respectivamente. A lista ordenada permite definir o tipo de sequencia por meio do parâmetro TYPE=[1,A,a,X,x]

Exemplo de lista ordenada

  1. Pêra
  2. Banana
  3. Uva
  4. Maçã
  5. Abacate

Exemplo de lista não ordenada

  • Carne
  • Cerveja
  • Linguiça
  • Pão de alho
